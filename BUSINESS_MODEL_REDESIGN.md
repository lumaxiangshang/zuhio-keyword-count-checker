## 🎯 新商业模式

### 定价策略

| 计划 | 价格 | 核心价值 | 目标用户 |
|------|------|----------|----------|
| **Free** | $0 | 每天 3 次分析 | 体验用户 |
| **Pro** | $9.99/月 或 $99/年 | 无限分析 + 高级功能 | 个人创作者 |
| **Business** | $29.99/月 或 $299/年 | 团队协作 + API | 小团队/机构 |

### Pro 版功能
- ✅ 无限关键词分析
- ✅ 高级 SEO 建议
- ✅ 30 天历史记录
- ✅ 导出 PDF/CSV 报告
- ✅ 多语言支持

### Business 版功能
- ✅ 所有 Pro 功能
- ✅ 无限历史记录
- ✅ 批量分析（100 篇/天）
- ✅ API 访问
- ✅ 团队共享（5 人账号）
- ✅ 优先支持

## 📊 用户流程

### 免费用户体验

```
访问首页 → 直接使用（无需登录）→ 剩余次数显示
                                      ↓
                              次数用完后
                                      ↓
                              弹出升级提示
                                      ↓
                              选择 Pro/Business
                                      ↓
                              PayPal 支付
                                      ↓
                              立即解锁无限使用
```

### 订阅用户流程

```
登录 → Dashboard → 查看使用统计
              ↓
         开始分析（无限次）
              ↓
         查看历史记录
              ↓
         导出报告
```

## 🛠️ 技术改动

### 数据库修改

1. **移除积分相关字段**
   - `User.credits` → 删除
   - `Payment.credits` → 删除
   - `Payment.paymentType` → 简化

2. **保留订阅相关**
   - `User.subscriptionPlan`
   - `User.subscriptionStatus`
   - `User.subscriptionEndDate`
   - `User.paypalSubscriptionId`

### API 修改

1. **删除积分 API**
   - `/api/paypal/create-order` → 改为创建订阅
   - `/api/paypal/capture-order` → 改为激活订阅

2. **新增订阅管理**
   - `/api/subscription/create` - 创建订阅
   - `/api/subscription/cancel` - 取消订阅
   - `/api/subscription/status` - 查询状态

### 前端修改

1. **首页**
   - 移除登录要求（先用后登）
   - 显示剩余免费次数
   - 用完后显示升级按钮

2. **Dashboard**
   - 显示订阅状态
   - 显示使用统计
   - 订阅管理入口

3. **定价页面**
   - 清晰的对比表格
   - 突出 Pro 版（Most Popular）
   - 年付优惠标识
